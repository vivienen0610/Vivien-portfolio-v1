# 如何自己更新這個網站

## 更新流程（4 步）
1. 想改什麼 → 用下方「貼給 AI 的範本」請任何 AI 幫你改
2. 把 AI 回傳的**完整程式碼**貼回 `index-styleA.html`（全選 → 取代）
3. 打開 **GitHub Desktop** → 填一句說明 → **Commit** → **Push origin**
4. 約 1 分鐘後 Netlify 自動上線 ✅

> 改壞不用怕：每次 Push 都是存檔點，可在 GitHub Desktop → History 還原到前一版。

---

## 📋 貼給 AI 的範本（複製下面整段）

```
我有一個單頁的個人作品集網站，全部內容都在一個 HTML 檔案裡（中英雙語）。
完整程式碼我會貼在最後。

請幫我做以下修改：
【在這裡用白話寫你想改什麼，例如：
 把聯絡資訊的電話改成 0912-345-678；
 或把某個作品卡片的標題改成「XXX」】

修改要求：
1. 請回傳「完整的」HTML 程式碼，不要只給片段，方便我整段取代。
2. 這是中英雙語網站，同一句話通常有中文與英文兩個版本
   （例如 <span class="zh">中文</span><span class="en">English</span>，
    或 T('中文','English')），請兩邊都一起更新。
3. 除了我指定的地方，其他內容、排版、樣式請保持原樣，不要自行更動或刪減。
4. 不要改動圖片路徑（images/…）。

以下是完整程式碼：
【貼上 index-styleA.html 的全部內容】
```

---

## 小提醒
- **所有內容都在 `index-styleA.html` 這一個檔**；找某段文字用 `Cmd/Ctrl + F` 搜尋那句中文。
- **換圖片**：AI 幫不了，要自己把新圖放進 `images/` 資料夾，檔名對到程式碼裡的路徑。
- 檔案太大貼不下時：只貼「要改的那一段」，並請 AI「只回那一段」，再貼回原位置。
- **不要**再用「拖曳 deploy 資料夾到 Netlify」的舊方法（會跟 Git 自動部署打架）。
